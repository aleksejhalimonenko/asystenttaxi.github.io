# asystenttaxi.github.io

Веб-приложение для учета расходов на автомобиль с фронтендом на HTML/CSS/JS и бэкендом на Google Apps Script.

Основной функционал:
📊 Главная панель с общей статистикой
⛽ История заправок и расход топлива
🔧 Учет сервисного обслуживания
💰 Отслеживание всех расходов
📱 Адаптивный мобильный интерфейс


Структура проекта
auto-service-tracker/
│
├── 🎨 Frontend/
│   ├── index.html          # Single Page Application
│   ├── style.css           # Стили и UI компоненты
│   ├── script.js           # Основная логика приложения
│   └── addFuel.js          # Форма добавления заправок
│
├── ⚙️ Backend (GAS)/
│   ├── код.gs              # 📖 Чтение данных (GET endpoints)
│   ├── add.gs              # 📝 Запись данных (POST endpoints)
│   └── calculations.gs     # 📊 Расчеты статистики
│
└── 📊 Data (Google Sheets)/
    ├── Главная             # Общая статистика и настройки
    ├── Топливо             # История заправок
    ├── Обслуживание и ремонт # Сервисные работы
    ├── Тюнинг и аксессуары # Доп. оборудование
    └── Штрафы и документы  # Документы и штрафы
    
Поток данных
Frontend (SPA)
    ↓
Google Apps Script API
    ↓  
Google Sheets (База данных)
    ↓
JSON Response → Frontend
